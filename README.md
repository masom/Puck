# Puck & Pixie
Puck is a virtualization configuration server used by Pixie to configure FreeBSD hypervisors and their jails.

It is not yet complete.

# Running Pixie Tests

`cd pixie`
`python -m unittest discover`

# External Requirements
- CherryPy 3.2
- Mako 0.5
